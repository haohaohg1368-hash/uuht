## 💡 下载技巧

### 方法一：直接从亚马逊网页下载

1. 打开产品页面
2. 将鼠标悬停在主图上
3. 右键 → "图片另存为"
4. 保存并重命名

### 方法二：使用浏览器开发者工具（获取高清大图）

1. 打开产品页面
2. 按 F12 打开开发者工具
3. 点击主图
4. 在 Elements 面板找到 `<img>` 标签
5. 复制 `src` 属性中的 URL
6. 在新标签页打开并下载

### 方法三：使用亚马逊产品 API（高级用户）

如果您熟悉编程，可以使用 Amazon Product Advertising API 获取高清产品图片。
