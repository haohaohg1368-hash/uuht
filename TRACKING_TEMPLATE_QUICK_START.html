步骤 LinkHaiTao 您的系统 (UUHT)
1. 设置追踪模板 {lpurl}?mmc=xxx&ranMID=xxx {lpurl}?utm_source=google&campaign={campaignid}
2. 设置最终网址 https://www.adidas.ae/en https://www.uuht.shop/go/air-fryer

• 您拥有完整的数据控制权
• 无需依赖第三方平台
• 可以自定义任意追踪参数
• 实时查看点击统计和转化数据

快速开始 (3 步完成)

步骤 1: 访问追踪模板生成器

等待 Vercel 部署完成后,访问:

https://www.uuht.shop/admin/tracking-template
打开追踪模板生成器 →

步骤 2: 生成追踪模板

  1. 选择要推广的短链接 (如 air-fryer)
  2. 选择快速模板或自定义参数
  3. 点击 "生成追踪模板"
  4. 复制生成的模板和最终到达网址

步骤 3: 在 Google Ads 中配置

  1. 登录 Google Ads
  2. 进入广告系列设置 → "广告系列网址选项"
  3. 粘贴追踪模板到 "跟踪模板" 字段
  4. 在广告的 "最终到达网址" 中输入短链接
  5. 保存并发布广告

💡 实际示例

场景: 推广 Amazon 空气炸锅

追踪模板:
{lpurl}?utm_source=google&utm_medium=cpc&utm_campaign={campaignid}&affiliate_id=YOUR_ID
最终到达网址:
https://www.uuht.shop/go/air-fryer
✅ 实际效果:
用户点击广告后:
1️ 跳转到: https://www.uuht.shop/go/air-fryer?utm_source=google&...
2️⃣ 系统记录点击和所有参数
3️ 自动跳转到: https://www.amazon.com/dp/B08N5WRWNW?tag=yourtag-20&utm_source=google&...
4️ 用户购买 → 您获得佣金 💰

📚 Google Ads 动态参数参考

  • {campaignid}广告系列 ID
  • {adgroupid}广告组 ID
  • {keyword}触发广告的关键词
  • {matchtype}匹配类型 (exact/phrase/broad)
  • {creative}广告创意 ID
  • {device}设备类型 (mobile/desktop/tablet)
  • {gclid}Google Click ID (用于转化追踪)
  • {lpurl}落地页网址 (自动替换为最终到达网址)
